This is a simple Caesar Cipher web application that allows users to encrypt and decrypt text using a shift value. The app provides an interactive UI to input a message, choose an operation (encryption or decryption), and specify the shift value. It then displays the transformed text.

How It Works:-
Select an operation: Choose between encryption or decryption.
Enter text: Provide the message you want to process.
Enter a shift value: Specify the number of shifts (integer).
Click "Submit": The transformed text will be displayed.
